Installation on Fedora systemsÂ¶

  • Fedora (20):

    sudo yum install make git python python-yaml python-matplotlib numpy scipy PyQt4
sudo easy_install progressbar
sudo easy_install pyavl
cd ~/src/   # or wherever you keep your source packages
git clone git://github.com/pyrocko/pyrocko.git pyrocko
cd pyrocko
sudo python setup.py install

For instructions on how to install Pyrocko on other systems or if the installation with the above procedure fails, see Quick system specific installation instructions or Detailed installation instructions.
